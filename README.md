# Visual Regression Testing Tool

## Funtion:

This script takes two images, mainly screenshots, and compares file 1 and file 2 and then it will darken to black the identical areas of said images and brighten or highlight the areas that are different.

## Use Case:

This can be used in a situation where you need to identify changes of a website, app or photo.
